## [1.1.0] · 2026-04-28 · 功能增强

### 新增
- **Logo 裁剪弹窗** — 上传 Logo 后弹出可交互裁剪框，支持拖动选区和四角缩放，确认后生成裁剪图像
- **二维码预览缩放** — 预览区新增 +/- 按钮，可在 160–520px 范围内调整预览尺寸（下载分辨率不变）
- **标签格式选项** — 文字标签新增粗体（B）、斜体（I）、阴影（影/Shd）三个格式开关，实时预览生效
- **标签默认字号调整** — 默认标签字号由 18px 改为 35px，开箱即用更清晰
- **默认简单模式** — 应用启动默认进入简单模式，降低初次使用门槛

## [1.0.0] · 2026-04-27 · Phase 3 完善与发布

### 新增
- 复制二维码图片到剪贴板（Clipboard API）
- 生成中的旋转加载动画
- 空状态提示（未输入内容时显示引导文字）
- 内容字节长度校验（超限红色提示，最大 1800 字节）
- SEO 基础设置（title、description、og meta）
- GitHub Actions 自动部署到 GitHub Pages
- Cloudflare Pages 支持（CF_PAGES 环境变量区分 base 路径）
- 中英双语界面（Language 切换）
- 简单 / 自定义模式切换（简单模式隐藏高级配置）

### 修复
- **中文乱码** — 通过手动 UTF-8 字节编码绕过 qrcode-generator 的 8-bit 截断缺陷
- **labelsInFrame 崩溃** — containerRef div 固定在 React 树同一位置，防止 canvas 被卸载
- **复制/下载圆角丢失** — compose canvas 时先填充对比色外框，圆角在任何背景下可见

### 优化
- 2x 清晰渲染（560px canvas CSS 缩放至 280px，retina 级别质量）
- Inter + Noto Sans SC 字体，中英数字全覆盖
- 软灰浅色主题（页面 #E8E8E8，卡片 #F5F5F5，不再纯白）
- 深色模式默认
- 模板选中状态高亮（双描边 + 阴影）
- 可调边框（内边距 + 圆角滑杆，标签入框选项）

## [0.2.0] · 2026-04-27 · Phase 2 样式定制

### 新增
- 点阵样式选择：方形 / 圆角 / 圆点
- 定位符（缩眼）样式选择：方角 / 圆角 / 圆点
- 中心 Logo 上传（本地图片，不上传服务器），上传时自动切换 H 级纠错
- Logo 大小滑块（20%–40%）
- 预设模板：极简黑 / 复古蓝 / 赛博粉
- 文字标签：上下左右四个位置，各自配置字体、字号、颜色，支持换行
- 深色模式切换
